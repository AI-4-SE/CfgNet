export(file: TextIO, include_unlinked: bool) → None¶

Export the network graph in the DOT graph description language.

Edges between linked configuration nodes will be drawn in red.

Parameters

  • file – file to export the network graph

  • include_unlinked – if true include all value nodes, else only linked nodes

class cfgnet.exporter.exporter.JSONExporter(network: Network)¶

Bases: object

export(file: TextIO, include_unlinked: bool)¶

Export the entire network graph to JSON, for visualizing with D3.js.

Links have a “type” property that is “link” for linked configuration nodes and “network” for links that constitute the network hierarchy.

Parameters

  • file – file to export the network graph

  • include_unlinked – if true include all value nodes, else only linked nodes
