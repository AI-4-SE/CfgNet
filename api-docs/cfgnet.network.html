cfgnet.network.ignorefile module¶

class cfgnet.network.ignorefile.IgnoreFile¶

Bases: object

Ignorefile functionality.

To ignore certain (types of) files or directories for network creation, a .gitignore-like ignorefile can be created at .cfgnet/ignore.

Each line of this ignorefile contains a glob-style pattern. Any files that match one or more of these patterns will be ignored by the tool.
